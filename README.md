  ## What is this?
  
  Oge Tibeb is a modern online shopping website for African clothing, designed for customers in the United States and Canada. The website showcases traditional African fashion items like Ankara dresses and other beautiful clothing pieces.
  
## Main Features:

### Shopping Catalog
- Browse through a collection of African clothing items
- View product images, names, prices, and descriptions
- Add items to your shopping cart with a single click
### Shopping Cart
- See all items you've selected in your cart
- Adjust quantities (increase or decrease) for each item
- Remove items you no longer want
- View the total cost of your order
- The cart remembers your items even if you navigate between pages
### Checkout Page
- Review your complete order before purchasing
- Enter shipping information (name, address, email, phone)
- The payment method is using a credit card.
- See order summary with total costs
### How It's Built
This website is built using modern web technologies that make it fast, responsive, and easy to use:
- React – A popular framework that makes the website interactive and smooth
- Tailwind CSS – A styling system that gives the website its clean, professional look
- Vite - A tool that makes the website load quickly

The website works on all devices – computers, tablets, and smartphones.
